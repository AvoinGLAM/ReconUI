Introduction

Welcome to this Wikidata reconciliation interface experiment!

Explore various ways to display Wikidata reconciliation candidates using rich data from different sources.

Currently, you can enter a search term and receive results from Wikidata's Action API via SPARQL. You can view them in Wikidocumentaries and on the map. Other interfaces are not working.

You cannot match data for real yet.

We hope to create an interface that can integrate with any data wrangling tool, read the items to be reconciled, and return the results in a simple format. Ideally, it could also integrate multiple search methods

Give it a try and share your feedback!

Help

To start searching, follow these steps:

  • Type a search term in the input box above and hit "Submit".
  • Navigate between candidates using the arrow keys.
  • Click on the dropdown menus to switch views or settings.
