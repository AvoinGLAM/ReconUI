Introduction

Welcome to this Wikidata reconciliation interface experiment!

Explore various ways to display Wikidata reconciliation candidates using rich data from different sources.

Currently, you can enter a search term and receive results from Wikidata's Action API via SPARQL. In the future, we aim to integrate multiple search methods.

We hope to create an interface that can integrate with any data wrangling tool, read the items to be reconciled, and return the results in a simple format.

Give it a try and share your feedback!

Help

To start searching, follow these steps:

  • Type a search term in the input box above and hit "Submit".
  • Navigate between candiiates using the arrow keys.
  • Click on the dropdown menus to switch views or settings.
