Introduction

Welcome to Eka – the Wikidata reconciliation app!

Explore various ways to display Wikidata reconciliation candidates using rich data from different sources.

Currently, you can enter a search term and receive results from Wikidata's Action API via SPARQL. You can view each candidate in all Wikimedia projects and on the map. Other interfaces are not available for the time being.

You cannot match data for real yet.

Eka will work with a spreadsheet, read the items to be reconciled, and return the matched Wikidata item's QID. Different search scenarios can also be developed, for example based on coordinates, authority ids or images, or using AI to find probable matches – and combinations of these.

Give it a try and share your feedback!

Help

To start searching, follow these steps:

  • Type a search term in the input box above and hit "Submit" or press "Enter".
  • Select a candidate to explore it in the two frames.
